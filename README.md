## git-tutorial
Toy shell for git tutorial purposes

Clone this repository by running the following command:

    git clone https://github.com/larics/git-tutorial-code

### C++
* Building:

      cd cpp
      mkdir build
      cmake ../
      make

* Running:

      ./lust

### Python

* Running:

      cd python
      ./lust.py
